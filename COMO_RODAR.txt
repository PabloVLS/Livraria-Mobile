================================================================================
TROUBLESHOOTING
================================================================================

PROBLEMA: "No connected devices!"
SOLUÇÃO:
1. Verificar se há emulador ativo no AVD Manager
2. Se não, clicar "Create Virtual Device"
3. Selecionar um dispositivo (Pixel 5 recomendado)
4. Clicar "Play" para iniciar
5. Aguardar o emulador carregar completamente
6. Tentar rodar novamente

PROBLEMA: "Build failed"
SOLUÇÃO:
1. Build → Clean Project
2. Build → Rebuild Project
3. Verificar se não há erros de sintaxe
4. Reiniciar Android Studio

PROBLEMA: "App fecha ao abrir"
SOLUÇÃO:
1. Verificar Logcat para erro específico
   View → Tool Windows → Logcat
2. Se for erro de View, verificar activity_main.xml
3. Se for erro de Adapter, verificar item_book.xml

PROBLEMA: "Emulador não inicia"
SOLUÇÃO:
1. AVD Manager → Editar o dispositivo
2. Mudar Graphics para "Software"
3. Aumentar memória para 1GB
4. Tentar novamente
